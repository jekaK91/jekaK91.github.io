Aimp - лучший аудиоплеер

  • Мультимедиа
  • Аудио

Aimp - это современный бесплатный аудиоплеер, разработан российскими программистами на языке программирования Delphi. Самое большое достоинство этого плеера, это поддержака большого количества форматов:
.CDA, .AAC, .AC3, .APE, .DTS, .FLAC, .IT, .MIDI, .MO3, .MOD, .M4A, .M4B, .MP1, .MP2, .MP3, .MPC, .MTM, .OFR, .OGG, .OPUS, .RMI, .S3M, .SPX, .TAK, .TTA, .UMX, .WAV, .WMA, .WV, .XM, .DSF, .DFF, MKA, AA3, AT3, OMA, YouTube.
Второе достоинство это фонотека. Фонотека представляет собой хорошо организованные музыкальные файлы, с легкостью упорядочивает по жанрам, годам, альбомам и авторам.
Дополнительных возможностей очень много, перечислять все нет смысла, вот одни из тех о который молчать не стоит.


Интернет-радио:
Прослушивание радиостанций в форматах OGG / WAV / MP3 / AAC / AAC+
Запись радиостанций в формат APE, FLAC, OGG, WAV, WV, WMA и MP3
Запись без потери качества для форматов MP3 / AAC / AAC+


Поддержка модулей расширения функциональности:
Можно добавлять новые утилиты и расширять уже имеющиеся.


Доступно большое количество обложек, аудиоконвертер, редактор тегов, создание закладок и очереди воспроизведения.
